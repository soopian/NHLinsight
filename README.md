NHL Insight Hub
Project Description

NHL Insight Hub is a web application that transforms raw NHL statistics into easy-to-understand player insights and comparisons. The system pulls live player statistics from the NHL API, calculates useful performance metrics, and presents the data visually through charts and comparison tools.

Users can:

Search NHL players
View player statistics and trends
Compare players side-by-side
Save favorite players
Analyze recent performance visually

The application solves the problem of scattered and difficult-to-interpret hockey statistics by turning raw data into meaningful insights.
