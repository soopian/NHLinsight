About This Project

NHL Insight Hub is a simple hockey dashboard built for an intro web development class. The frontend uses FetchAPI to request data from backend routes. The backend connects to the NHL API for hockey data and Supabase for saved favorite teams.


Technologies Used

HTML, CSS, JavaScript, Express, NHL API, Supabase, Chart.js, and AOS animations.


Frontend Libraries

Chart.js is used for a standings chart. AOS is used for simple fade-in animations.
